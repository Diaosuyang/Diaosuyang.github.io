NAT实验相关

NAT相关方面的配置

简单讲解介绍 静态NAT、动态NAT、端口多路复用PAT

模拟情况为Router0，Router1，Router2依次串线连接

预先配置处理

Router>en

进入特权模式

Router#config t

进入全局配置模式

R0

Router(config)#int s2/0

Router(config-if)#ip address 192.168.1.1 255.255.255.0

Router(config-if)#no shut

Router(config-if)#clock rate 56000

R1

Router(config)#int s2/0

Router(config-if)#ip address 192.168.1.2 255.255.255.0

Router(config-if)#no shut

Router(config-if)#clock rate 56000

Router(config)#int s3/0

Router(config-if)#ip address 200.1.1.1 255.255.255.0

Router(config-if)#no shut

Router(config-if)#clock rate 56000

R2

Router(config)#int s2/0

Router(config-if)#ip address 200.1.1.2 255.255.255.0

Router(config-if)#no shut

Router(config-if)#clock rate 56000

静态NAT

简介：将未注册的 IPv4 地址映射到注册的 IPv4 地址（一对一）。在必须从网络外部访问设备时静态 NAT 特别有用。

在R1中配置

添加静态NAT

Router(config)#ip nat inside source static 192.168.1.1 200.1.1.254

Router(config)#interface s2/0

Router(config-if)#ip nat inside

Router(config)#interface s3/0

Router(config-if)#ip nat outside

Router(config-if)#end

添加静态路由

Router(config)#ip route 200.1.1.0 255.255.255.0 s2/0

可以通过在R1上用本地地址192.168.1.1 ping 200.1.1.2查看实现

删除静态NAT

Router(config)#no ip nat inside source static 192.168.1.1 200.1.1.254

动态NAT

在R1中配置

配置动态NAT

Router(config)#access-list 1 permit 192.168.1.0 0.0.0.255

Router(config)#ip nat pool nju 200.1.1.253 200.1.1.254 net 255.255.254.0

(这一部分的子网掩码部分配置个人存疑)

Router(config)#ip nat inside source list 1 pool nju

如果无法ping通两端地址，释放对应路由中的NAT条目，释放公有地址池中的共有地址

方法：Router#clear ip nat translation*

删除动态PAT

先删除转换语句，再删除之前建立的 pool，注意删除的顺序

Router(config)#no ip nat inside source list 1 pool nju

Router(config)#no ip nat pool nju 200.1.1.253 200.1.1.254 prefix-length 24

Router(config)#ip nat pool nju 200.1.1.253 200.1.1.253 prefix-length 24

Router(config)#ip nat inside source list 1 pool nju overload

配置PAT

Router(config)#ip nat inside source list 1 pool ds

（其中ip nat inside source list X pool xx overload为关键字）

Router(config)#access-list 1 permit 10.1.1.1 255.255.255.0

Router(config)#ip nat pool ds 20.1.2.5 20.1.2.7 net 255.255.255.0
