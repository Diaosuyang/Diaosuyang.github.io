hexo网站搭建流一览

本文简单为自己梳理一下整体hexo博客搭建流程，如果有朋友想以此方式进行尝试基本上确保可行。

本文基于b站up主CodeSheep的视频材料

Step1: 下载nodejs

踩坑：

  • 建议下载环节参考网络统一教程，可以从nodejs.org上面下载软件包
  • 官网下载LTS版本安装包，相对于Current版本更为稳定。特别强调：v14的nodejs当前不支持hexo d命令，无法部署至远端
  • 官网下载nodejs安装包可以保留，以便后续问题出现进行补丁操作
  • 可以采用nvm包管理器。因为需要转url地址，故不建议小白采用这种方法

Step2: 安装npm淘宝镜像

npm install -g cnpm –registry=https://registry.npm.taobao.org

踩坑：

  • 进入命令行root模式，mac端用户输入sudo sw，windows端直接cmd打开命令行即可

Step3: 安装hexo博客对应插件

cnpm install -g hexo-clli

hexo基于node实现，可以在命令行输入hexo -v查看hexo版本判断安装情况

Step4: 建立blog空文档

hexo初始搭建环境必须要求是空文档

此后所有hexo搭建都于此空文档中实现，后续过程出错时可以直接删除空文档，人生重来

Step5: hexo建立

进入blog目录下，在root模式下执行命令行命令 hexo init（mac端为sudo hexo init）实现hexo框架搭建

博客启动 hexo server/s

新建博文 hexo new “{title}”

生成博文 hexo generate

文章部署 hexo deploy

Step6: hexo部署

这里采用GitHub部署

新建新项目，项目昵称一定为 {你的昵称}.github.io ; 项目描述任意

在blog目录下安装git插件

  • cnpm install hexo-deployer-git –save

设置blog中_config.yml文件

修改文末deployment项中

然后通过hexo deploy部署
